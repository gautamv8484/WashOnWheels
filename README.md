## 🎯 About

**WashOnWheels** is a full-stack MERN application for an **on-demand doorstep vehicle cleaning service**. Customers can book professional car/bike washing services at their home or office, subscribe to monthly plans, and track their service history — all from one platform.

### 💡 Core Idea
- ❌ No need to visit washing centers
- ⏱️ Save time with doorstep service
- 🌱 Eco-friendly waterless cleaning
- 💰 Affordable subscription plans
- 📱 Easy online booking

## ✨ Features

### 👤 User Features
| Feature | Description |
|---------|-------------|
| 🔐 Authentication | Register, Login, Protected routes |
| 🚗 Booking System | 3-step interactive booking form |
| 💳 Subscription Plans | Basic, Premium, Business plans |
| 📊 User Dashboard | View bookings, plans, history |
| ⏳ Days Remaining | Visual progress bar for active plans |
| 🔄 Plan Renewal | Renew or reactivate expired plans |
| 📋 Complaint System | Report issues with priority levels |
| 👤 Profile Management | Update personal information |
| 📱 Responsive Design | Works on all devices |

| Feature | Description |
|---------|-------------|

| 👑 Manage Subscriptions | Track all active/expired plans |
| 📨 Contact Messages | View and respond to inquiries |
| ⚠️ Complaints Panel | Manage user complaints by priority |

| 🔐 Admin Authentication | Secure admin login |

1. Clone Repository
git clone https://github.com/gautamv8484/washonwheels.git
cd washonwheels

2. Setup Backend

# Navigate to server
cd server

# Install dependencies
npm install

# Create .env file
cp .env.example .env

# Edit .env with your values
# Add MongoDB URI, JWT Secret etc.

# Start server
npm run dev
3. Setup Frontend

# Open new terminal
cd client

# Install dependencies
npm install

# Start development server
npm run dev
4. Open Application
text

Frontend  → http://localhost:5173
Backend   → http://localhost:5000
Admin     → http://localhost:5173/admin/login
